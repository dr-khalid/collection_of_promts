# Collection of Prompts

A personal collection of prompts, skills, and agent files for AI workflows.

## Contents

- **Prompts** — Reusable prompt templates for various tasks
- **Skills** — Modular skill definitions for AI agents
- **Agents** — Agent configuration and instruction files

## Usage

Browse the folders to find prompts or agents relevant to your use case. Each file is self-contained and can be copied or adapted freely.
